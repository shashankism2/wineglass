The General Root Calculator allows you to find the nth root of any positive or negative number. Whether you need square roots, cube roots, or higher-order roots, this tool quickly calculates accurate results using the standard mathematical root formula.

What is an nth Root?

The nth root of a number a is a value that, when raised to the power of n, gives the number a. This is useful in algebra, geometry, physics, engineering, and complex number operations.

General Root Formula

n√a = a1/n

Where:

  • a = the number you want the root of
  • n = the degree of the root (2 for square root, 3 for cube root, etc.)
  • a1/n = exponent form used for calculation

Examples

Example 1: Square Root

Find the square root of 49.
n = 2
²√49 = 491/2 = 7

Example 2: Cube Root

Find the cube root of 64.
n = 3
³√64 = 641/3 = 4

Example 3: 4th Root

Find the 4th root of 16.
n = 4
⁴√16 = 161/4 = 2

Example 4: Root of a Decimal

Find the cube root of 0.008.
n = 3
³√0.008 = 0.0081/3 = 0.2
