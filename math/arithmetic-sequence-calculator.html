The Arithmetic Sequence Calculator helps you find terms of an arithmetic sequence (also called an arithmetic progression) using the first term, common difference, and the position of the term. In an arithmetic sequence, each term increases or decreases by a constant value known as the common difference (d).

The general formula for the nth term of an arithmetic sequence is:

an = a1 + (n − 1)d

  • an — the nth term of the arithmetic sequence
  • a1 — the first term of the sequence
  • n — the position of the term you want to find
  • d — the common difference between consecutive terms

This calculator instantly computes any term in the sequence and helps you understand patterns, progressions, and number growth.

Examples

  • Example 1:
    First term (a₁) = 3,   Common difference (d) = 2,   n = 5
    Result: a₅ = 3 + (5 − 1) × 2 = 11

  • Example 2:
    First term (a₁) = 10,   Common difference (d) = −3,   n = 4
    Result: a₄ = 10 + (4 − 1) × (−3) = 1

  • Example 3:
    First term (a₁) = -5,   Common difference (d) = 7,   n = 6
    Result: a₆ = -5 + (6 − 1) × 7 = 30
